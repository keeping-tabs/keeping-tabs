Welcome to Keeping Tabs!

Keeping Tabs is a Chrome Extension that manages idle tabs, closing them when inactive for a prolonged period of time. If you've ever had an issue with minding your tabs, then this is the extension for you! Keeping Tabs will close tabs based on specific settings you control, and store them for later perusal. View and download the source code from GitHub.

Load the extension

  • Head to the top right corner of your Chrome browser!
  • Click Settings
  • On the sidebar, choose extensions
  • Ensure that the Developer mode checkbox in the top right-hand corner is checked
  • Click Load unpacked extension to pop up a file-selection dialog
  • Navigate to the directory in which your extension files live, and select it
Keeping Tabs is maintained by keeping-tabs.
